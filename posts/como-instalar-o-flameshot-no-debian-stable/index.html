Como instalar o flameshot no Debian stretch

Sumário

  • Referências

O flameshot é um excelente utilitário para tirar screenshots e já realizar as anotações no mesmo momento.

Dica

Atualização: Agora que o Debian 10 foi lançado não é mais necessário recorrer ao backports, então basta usar apt install flameshot.

O flameshot já está disponível no Debian 10, mas enquanto aguardamos a versão nova do Debian ser lançada é preciso utilizar a versão do stretch-backports.

Primeiramente vamos acrescentar o repositório backports:

1echo "deb http://deb.debian.org/debian stretch-backports main" \
2     >> /etc/apt/sources.list.d/backports.list

Atualizar a lista de pacotes disponíveis:

1apt update

Instalar usando a opção -t stretch-backports para indicar que deve ser utilizado o repositório backports:

1apt install -t stretch-backports flameshot
