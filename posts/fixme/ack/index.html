ack - Razões para usar para código fonte

Sumário

  • Instalação
  • Configuração
  • Ignora os diretórios VCS e arquivos de backup
  • É mais fácil restringir a busca por tipo
  • Criação de lista de arquivos sem busca
  • Expressão regular do perl
  • Referências

Instalação

1brew install ack
2apt install ack-grep

Configuração

O arquivo de configuração do ack é o ~/.ackrc:

 1# Configuração do ack
 2
 3# Só faz busca sensível para mixed case
 4--smart-case
 5
 6# Segue os links simbólicos
 7--follow
 8
 9--type-add=css=.sass,.less,.scss
10--type-add=ruby=.rake,.rsel,.builder,.thor
11--type-add=html=.haml,.html.erb,.html.haml
12--type-add=js=.js.erb,.coffee
13--type-set=cucumber=.feature
14--type-set=c=.c,.cpp,.ino,.pde,.h
15
16# add specific type for haml
17--type-set
18haml=.haml
19
20# add specific type for sass
21--type-set
22sass=.sass,.scss
23
24# add specific type for coffee
25--type-set
26coffee=.coffee
27
28--ignore-dir=vendor
29--ignore-dir=log
30--ignore-dir=tmp
31--ignore-dir=doc
32--ignore-dir=coverage
33--ignore-dir=public/assets
34--ignore-dir=spec/cassettes
35--ignore-dir=features/cassettes
36
37--sort-files
38
39# opções que impedem o vim de funcionar corretamente
40# --group
41# --color
42
43# vim: ft=conf

Ignora os diretórios VCS e arquivos de backup

ignora:

  • foo~ e #foo#
  • arquivos binários
  • core dumps
1grep pattern $(find . -type f | grep -v '\.git')
2
3ack pattern

É mais fácil restringir a busca por tipo

1grep pattern $(find . -name '*.pl' -or -name '*.pm' -or -name '*.pod' | grep -v .git)
2
3ack --perl pattern

Criação de lista de arquivos sem busca

1# List all Ruby files in the tree
2ack -f --ruby > all-ruby-files

Expressão regular do perl

1ack --cpp '#include\s+<(.*)>' --output '$1' -h
