Firefox

Importando certificado raiz de CAcert

Acesse o website do CAcert em http://www.cacert.org/index.php?id=3 e baixe o certificado em formato PEM.

No firefox acesse Preferências (1), Avançado (2) e Certificados (3).

Clicar em Ver certificados (4) e depois em Importar (5).

Selecione o root.pem baixado no passo anterior.

Clique no botão Ver (1) e verifique que a assinatura:

SHA1 Fingerprint: 135C EC36 F49C B8E9 3B1A B270 CD80 8846 76CE 8F33

Clique em fechar e depois em OK (2).

Google Chrome no Linux

In Linux, Google Chrome uses Mozilla's NSS for the certificates, then you need the certutil tool to manage it.

In Debian/Ubuntu certutil comes from libnss3-tools

$ sudo apt-get install libnss3-tools

and to import the our root certs you simply need to run:

$ cd ~/Documents/cacert/
$ wget -O cacert-root.crt "http://www.cacert.org/certs/root.crt"
$ wget -O cacert-class3.crt "http://www.cacert.org/certs/class3.crt"

$ certutil -d sql:$HOME/.pki/nssdb -A -t TC -n "CAcert.org" -i cacert-root.crt
$ certutil -d sql:$HOME/.pki/nssdb -A -t TC -n "CAcert.org Class 3" -i cacert-class3.crt

and now it just works, without needing to restart the browser.
