Or Equals

Operadores ||= ou or=

1value = 0  # Qualquer valor falso: 0, null, false,  ""
2value ||= "Novo valor"   # era falso e aceita novo valor
3value ||= "Outro valor"  # é verdadeiro e não atribui
4console.log value
5# Novo valor

Existencial

O operador existencial ?= é ativado apenas por undefined ou null.

1value ?= {}

Destructuring assignments

Não sei o que está sendo destruído!

Este operador copia os elementos para o destino na mesma ordem.

1someObject = { a: 'value for a', b: 'value for b' }
2{ c, d } = someObject
3console.log "a is '#{c}', b is '#{d}'"
4# a is 'value for a', b is 'value for b'
