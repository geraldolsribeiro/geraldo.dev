FCGI no Apache

Sumário

Rodando como servidor interno

O número de processos depende da capacidade da máquina, mas vamos manter em 1.

1# /etc/apache2/conf.d/your_wiki
2<IfModule mod_fastcgi.c>
3   AddHandler fastcgi-script .fcgi .fcg
4   FastCgiServer /your/path/to/moin.fcg -idle-timeout 60 -processes 1
5   ScriptAlias /your_wiki "/your/path/to/moin.fcg"
6</IfModule>

Ativação do módulo

1apt-get install libapache2-mod-fastcgi
2a2enmod fastcgi
3apache2ctl graceful

Parâmetros do FastCgiServer

-port [port]              - The port the application will use for communication with the web server.
-idle-timeout [seconds]   - The number of seconds of inactivity allowed before request is aborted.

Rodando como um servidor externo

Selecione um número de porta para comunicação interna. Ela deve ser maior que 1023. Para este exemplo vamos escolher 8888.

# httpd.conf
Alias /moin.fcg "/your/path/to/moin.fcg"
FastCgiExternalServer "/your/path/to/moin.fcg" -host localhost:8888

Parâmetros do FastCgiExternalServer

-host [hostname:port]     - The port and on what host name to respond.
-idle-timeout [seconds]   - The number of seconds of inactivity allowed before request is aborted.

Fallback para CGI se o FCGI não estiver disponível

<IfModule !mod_fastcgi.c>
    ScriptAlias /mywiki "/your/path/to/moin.cgi"
</IfModule>
<IfModule mod_fastcgi.c>
    AddHandler fastcgi-script .fcg
    ScriptAlias /mywiki "/your/path/to/moin.fcg"
</IfModule>
