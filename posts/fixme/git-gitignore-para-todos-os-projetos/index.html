gitignore para todos os projetos

Sumário

Configure ~/.gitignore e todos os projeto utilização este default.

Exemplo:

*.lock
*.log
*.pyc
*.sw[nop]
.DS_Store
.bundle
node_modules
public
tmp
vendor
