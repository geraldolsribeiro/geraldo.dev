URL da página

ADD !EXTRACT {{!URLCURRENT}}

Ocultar padrão obtido

Por padrão o texto extraído é mostrado na tela, para desabilitar isto use:

SET !EXTRACT_TEST_POPUP NO

Salvar com CSV

SAVEAS TYPE=EXTRACT

a saída será do tipo

"item1", "item2", "item 3", ...

Salvando valores arbitrários

ADD !EXTRACT <value>
ADD !EXTRACT {{!NOW:yyyy/mm/dd_hhnn}}

Extraindo de SELECT

Somente o valor ativo

TAG POS=1 TYPE=SELECT ATTR=TXT:*&&NAME:quantity&&VALUE:* EXTRACT=TXT

Todos os valores

TAG POS=1 TYPE=SELECT ATTR=TXT:*&&NAME:quantity&&VALUE:* EXTRACT=TXTALL

!EXTRACT contém a lista com todas as entradas, separadas pela palavra chave [OPTION]

Extração relativa

' Encontre o item de referência, por exemplo uma linha da tabela
TAG POS=1 TYPE=A ATTR=TXT:10030 EXTRACT=TXT
' Extrair valore relativos ao item anterior
' Use *,00 já que todos os valores, neste caso, terminam por ,00
TAG POS=R1 TYPE=A ATTR=TXT:*,00 EXTRACT=TXT
TAG POS=R2 TYPE=A ATTR=TXT:*,00 EXTRACT=TXT
' Posicionamento negativo
TAG POS=R-1 TYPE...

Loop

'Anchor to the exchange rates table
TAG POS=2 TYPE=TABLE ATTR=TXT:<SP><SP><SP><SP>USD<SP><SP><SP><SP>GBP*
'Extract each of the rows *inside* the table
SET !LOOP 3
TAG POS=R{{!LOOP}} TYPE=TR ATTR=TXT:* EXTRACT=TXT
