Performance web

Sumário

  • Apache Bench
  • Locust
  • Referências

A performance afeta o negócio de uma forma muito forte1.

Segundo o Website Response Times os tempos de resposta possuem os seguintes limites;

  • até 0.1 segundo: sensação que foi instantâneo e mantém um alto nível de responsividade
  • entre 0.1 e 0.3 segundo: o usuário consegue perceber o atraso
  • 1 segundo: limite para que o usuário não sinta interrupção no fluxo
  • 2 segundos: expectava de carregamento do site
  • maior que 3 segundos: 40% do usuário abandonam o site
  • 10 segundos: limite máximo que o usuário consegue manter a atenção

Moral da história, você pode perder metade dos clientes com um site lento.

Apache Bench

Quando se tem um único endpoint para testar é uma excelente alternativa.

Locust

A locust.io é uma ferramente de teste free, onde o código de teste é definido em python.

A instalação é via pip:

1pip install locustio

Exemplo:

 1from locust import HttpLocust, TaskSet, task
 2
 3class WebsiteTasks(TaskSet):
 4    def on_start(self):
 5        self.client.post("/login", {
 6            "username": "test_user",
 7            "password": ""
 8        })
 9
10    @task
11    def index(self):
12        self.client.get("/")
13
14    @task
15    def about(self):
16        self.client.get("/about/")
17
18class WebsiteUser(HttpLocust):
19    task_set = WebsiteTasks
20    min_wait = 5000
21    max_wait = 15000

Execução:

1locust -f locustfile.py
