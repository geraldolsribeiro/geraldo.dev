Publicação local do imbok

Sumário

  • Script para monitoramento
  • Configuração do apache
1ssh root@ethon
2cd /opt/samba/
3git clone ssh://git@intmain.io:8322/Intmain/imbok.git
4npm install gitbook-cli -g

Script para monitoramento

 1#!/bin/bash
 2
 3BRANCH=master
 4SLEEP=300
 5
 6while [ 1 ]
 7do
 8    if git remote -v update 2>&1 | grep $BRANCH | grep "up to date" &> /dev/null
 9    then
10        echo "Nada mudou!"
11    else
12        echo "Atualizando..."
13        git checkout $BRANCH
14        git pull origin $BRANCH
15        make html
16    fi
17    sleep $SLEEP
18done

Configuração do apache

 1cat <<EOF > /etc/apache2/sites-available/imbok.conf
 2<Directory /opt/samba/imbok/_book>
 3    AllowOverride all
 4    Options Indexes FollowSymLinks
 5    Require all granted
 6</Directory>
 7
 8Alias "/imbok" "/opt/samba/imbok/_book"
 9
10# vim: syntax=apache ts=4 sw=4 sts=4 sr noet
11EOF

Ativar a configuração

a2ensite imbok
systemctl reload apache2
