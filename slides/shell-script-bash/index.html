BASH - Bourne Again Shell

BASH - Bourne Again Shell

  • Default na maioria das distribuições Linux

Teclas de atalho (básico)

  • ^E: Vai para o final da linha
  • ^A: Vai para o início da linha
  • ^R: Busca por comando no histórico
  • ^L: Limpa a tela
  • ^C: Interrompe o processo atual
  • ^Z: Suspende o processo atual (bg e fg)
  • ^S: Suspende a saída para a tela
  • ^Q: Continua a saída para a tela

Teclas de atalho (todas)

1bind -P
