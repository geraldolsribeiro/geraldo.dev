Como limpar o audio com audacity

Last uptate: 15-May-2019
audacity

Efeito redução de ruído

  1. Selecionar somente o ruído (sem vóz)
  2. Menu Efeitos
  3. Menu Redução de ruídos
  4. Clicar no botão Obter perfil do ruído
  5. Ctrl+A Para para selecionar tudo
  6. Menu Efeitos
  7. Menu Redução de ruídos
  8. Clicar no botão OK

Efeito compressor

  1. Ctrl+A para selecionar tudo
  2. Menu Efeitos
  3. Menu Compressor
  4. Não precisa mexer em nada
  5. Clicar no botão OK

Efeito normalizar

  1. Ctrl+A para selecionar tudo
  2. Menu Efeitos
  3. Menu Normalizar
  4. Não precisa mexer em nada
  5. Clicar no botão OK

Efeito equalização

  1. Ctrl+A para selecionar tudo
  2. Menu Efeitos
  3. Menu Equalização
  4. Selecione a curva 100Hz Rumble
  5. Clicar no botão OK
