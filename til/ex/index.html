Editando arquivos com ex / vim

Last uptate: 15-Sep-2020
exvimbash
ex foo.txt <<-EOF  g/^\s*$/d  %s/^/[/  %s/\(.*\)\(\s\+\)\(.*\)/\3\2\1  %s/$/]/  %s/design/test/  wq " Update changes and quit.EOF
  • o comando ex é equivalente a vim -E.
  • acrescente -V1 para saída mais verbosa

Exemplo uma linha:

ex +"g/^\s*$/d" +"%s/^/[/" +"%s/design/test/" -cwq foo.txt
