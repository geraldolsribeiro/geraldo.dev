Liberando o alt do xfce para o figma

Last uptate: 26-Jul-2020
xfcefigma

Introdução

O Figma utiliza a tecla alt para copiar elementos e o xfce4 também utiliza a mesma tecla para mover janelas.

Alterando comportamento do alt pela interface gráfica

Para liberar a tecla Alt para uso pelo Figma acesse o painel de Configurações e depois vá em Ajustes do gerenciador de janelas:

Na aba Acessibilidade escolha None para a tecla de captura de janelas:

Alterando comportamento do alt pelo terminal

xfconf-query -c xfwm4 -p /general/easy_click -s none

Para retornar o Alt use:

xfconf-query -c xfwm4 -p /general/easy_click -s Alt
