git diff para odt

Last uptate: 20-Aug-2020
git

Instalar as dependências:

apt install odt2txt

Acrescentar o mapeamento das extensões no .gitattributes que serão conhecidas por odf:

*.ods diff=odf
*.odt diff=odf
*.odp diff=odf

*.ods difftool=odf
*.odt difftool=odf
*.odp difftool=odf

Definir qual ferramenta será usada para conversão de texto para o tipo odf.

git config diff.odf.textconv odt2txt
