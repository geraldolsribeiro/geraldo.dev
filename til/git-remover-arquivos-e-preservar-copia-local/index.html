Como remover arquivos do controle de versão preservando cópia local

Last uptate: 21-Jul-2021
git

As vezes é necessário remover arquivos apenas do controle de versão preservando cópia local do arquivo.

  1. Acrescente o arquivo ao .gitignore
  2. Remova do git com git rm --cached nome/do/arquivo

A flag --cached vai manter a cópia local.
