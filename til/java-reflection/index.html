Last uptate: 23-Nov-2019
java

SomeClass.class obtém o tipo Class<SomeClass> que é um objeto que representa o nome da classe em tempo de execução e é o mesmo objeto retornado pelo getClass.

NombreClase obj = new NombreClase();
System.out.println(NombreClase.class.getName());
System.out.println(obj.getClass().getName())
