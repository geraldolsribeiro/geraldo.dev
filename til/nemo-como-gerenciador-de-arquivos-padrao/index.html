Nemo como Gerenciador De Arquivos Padrão

Last uptate: 23-Jul-2019
file managernemonautilusgnomexfce

Trocando o Nautilus pelo Nemo no GNOME

xdg-mime default \  nemo.desktop inode/directory \  application/x-gnome-saved-search

Desabilitando o Nautilus

gsettings set org.gnome.desktop.background show-desktop-icons false

Habilitando o Nemo

gsettings set org.nemo.desktop show-desktop-icons true

Trocando o Thunar pelo Nemo no XFCE

Em Configurações acesse Aplicativos preferidos

Abra a aba Utilitários e altere para nemo em Gerenciador de Arquivos

Testando

xdg-open $HOME

Configurando o terminal

Usar o xfce4-terminal no nemo

gsettings set org.cinnamon.desktop.default-applications.terminal exec xfce4-terminal

Usar o gnome-shell no nemo

gsettings set org.cinnamon.desktop.default-applications.terminal exec gnome-shell

Usar o gnome-terminal no nemo

gsettings set org.cinnamon.desktop.default-applications.terminal exec gnome-terminal
